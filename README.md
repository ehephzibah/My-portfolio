# Personal Portfolio Website

A modern, responsive personal portfolio website built using **HTML**, **CSS**, and **JavaScript**. This portfolio showcases my projects, technical skills, education, achievements, and contact information in a clean and professional design.

##  Live Demo

 **Portfolio:** https://hephzibaheportfolio.netlify.app/



## Features

- Modern and responsive design
- Smooth scrolling navigation
- Hero section with introduction
- Featured projects showcase
- About section
- Skills section
- Learning journey
- Experience & activities
- Achievements
- Contact section
- Resume download
- Mobile-friendly layout
- Clean UI with reusable components
